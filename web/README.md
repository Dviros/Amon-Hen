# Amon Hen Web

Astro site for Amon Hen.

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Deploy

```bash
npm run deploy
```

Wrangler reads the Worker name and custom domain from `wrangler.jsonc`. Cloudflare credentials must come from the environment and must not be written into this repository.
